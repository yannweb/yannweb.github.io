别人的
1.解压tar包
cd /tmp
tar -xzvf mysql-5.6.21-linux-glibc2.5-x86_64.tar.gz
mv mysql-5.6.21-linux-glibc2.5-x86_64 mysql-5.6.21

2.添加用户与组
groupadd mysql
useradd -r -g mysql mysql
chown -R mysql:mysql mysql-5.6.21
yum install libaio
mv mysql-5.6.21/ /var

3.安装数据库
su mysql
cd /var/mysql-5.6.21/scripts
./mysql_install_db –user=mysql –basedir=/var/mysql-5.6.21 –datadir=/var/mysql-5.6.21/data
exit

4.配置文件
cd /var/mysql-5.6.21/support-files
cp my-default.cnf /etc/my.cnf
cp mysql.server /etc/init.d/mysql
vi /etc/init.d/mysql #若mysql的安装目录是/usr/local/mysql,则可省略此步
修改文件中的两个变更值
basedir=/var/mysql-5.6.21
datadir=/var/mysql-5.6.21/data

5.配置环境变量
vi /etc/profile
export MYSQL_HOME=”/var/mysql-5.6.21”
export PATH=”$PATH:$MYSQL_HOME/bin”
保存退出
. /etc/profile

6.添加自启动服务
chkconfig –add mysql
chkconfig mysql on

7.启动mysql
service mysql start

8.登录mysql及改密码与配置远程访问
mysqladmin -u root password ‘root’ #修改root用户密码
mysql -u root -p #登录mysql,需要输入密码
GRANT ALL PRIVILEGES ON . TO ‘root’@’%’ IDENTIFIED BY ‘root’ WITH GRANT OPTION;

#允许root用户远程访问
FLUSH PRIVILEGES;

#刷新权限
mysql>exit

由移动定制手机看待产品兼容性

Veröffentlicht am 2015-01-22   |   in 产品体验   |  

移动定制的M812手机是一款性价比很高的手机, 屏幕大,像素高. 但是xp环境下使用会有一点困难.
用户会发现插入usb线连接电脑时,sd卡无法被识别成为u盘,会报MTP driver驱动故障,从而安装失败.

MTP 是媒体传输协议, 一个win7自带的协议. 比较坑的是:xp不会自动升级安装这个协议,哪怕你一直升级也没用.倒是安装windows media player 11 是会顺带安装到电脑里(想想多少人从未打开过windows media player). 最后结果就是win7用户完全没有这方面的问题, 部分xp用户却完全使用不了.

我相信移动推出这款手机的时候, 肯定做过不少测试, 但重心多半在win7,win8上面, 至于xp嘛, 一个老系统, 微软都不支持了, 随便测测能跑就行了. 而事实恰恰相反, 就数量上而言, 新型号电脑相对较少, 而老电脑,老系统往往比预料到的多. 大量基数遇上删节版测试,问题就这样发生了.

给高效Todo的一些建议

Veröffentlicht am 2015-01-19   |   in 产品体验   |  

高效Todo是安卓上的一款效率软件, 界面简明,很受欢迎,也是我手机里的长驻程序.
多次使用后,感觉有一些小小的细节需要改进:

1 重复的任务开启时确认按钮位置
重复的任务,完成时会确认下一次会重复要不要执行. 但确认按钮在左边,取消在右边.
考虑大部分人都是右手便利,也就是左手拿手机,右手操作,很容易误按到取消键
而且,重复的任务,再确认,本来就是一件很累赘的事.

2 任务标记相关
高效Todo支持次级任务,也就是一个任务下,可以再建子任务,
但是,有子任务和没有子任务的图标是相同的,很容易漏掉子任务,建议在前面增加一个+号以示区别
而且, 次级任务和主任务没有从属关系, 次级全部完成以后,主任务也不会显示完成,需要多点一次完成

3 与电脑同步
高效Todo提供了账号同步功能,可以在两台手机之间同步, 应该是考虑了团体协作方面,毕竟带2部安卓手机的人不多,
但没有发现和电脑同步的方法,我觉得与电脑同步是个很重要的功能,毕竟手机打字也不是哪么舒服的,
最开心的还是在电脑上列好一周计划,然后上传到手机上,不停的 x x x x…
