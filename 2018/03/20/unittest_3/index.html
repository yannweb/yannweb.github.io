各种断言方法

断言方法检查你认为应该满足的条件是否确实满足, 如果满足则测试通过, 否则会引发异常

语句 断言
assertEqual(a,b) ? a == b
assertNotEqual(a,b) ? a != b
assertTrue(x) ? x = True
assertFalse(x) ? x = False
assertIn(item,list) item in list
assertNotIn(item, list) item not in list

虽然这个表格到处都有,还是手抄了一遍
